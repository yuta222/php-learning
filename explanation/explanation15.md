# 第15題 解説: メール文を作る

## 考え方

文章の一部だけを変えたいときは、変数を使います。
宛名、イベント名、日付を変数にしておくと、同じ形の文章を簡単に作れます。

## 手順

1. 宛名を `$to` に入れます。
2. イベント名を `$event` に入れます。
3. 日付を `$date` に入れます。
4. それぞれを文章の中で使います。

## つまずきポイント

ダブルクォートの文字列では、`{$event}` のように変数を入れられます。
日本語の文章と変数を混ぜるときは、`{}` を使うと読みやすいです。

## もう一度練習

イベント名を「PHP発表会」に変えて、文章全体が自然に変わるか確認しましょう。
