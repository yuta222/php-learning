# 第19題 解説: カウントダウン

## 考え方

`while` は、条件が正しい間だけ処理を繰り返します。
今回は、数が1以上の間だけ表示します。

## 手順

1. `$count` を5にします。
2. `$count >= 1` の間だけ繰り返します。
3. 表示したあと、`$count--` で1減らします。
4. 繰り返しが終わったら「スタート」と表示します。

## つまずきポイント

`$count--` を忘れると、ずっと同じ数字のままになり、繰り返しが終わりません。
`while` では、条件がいつか false になるように書きましょう。
