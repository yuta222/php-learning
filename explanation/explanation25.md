# 第25題 解説: 在庫表

## 考え方

商品名と在庫数の組み合わせは、連想配列で表せます。
キーを商品名、値を在庫数にすると、一覧表示が簡単になります。

## 手順

1. 商品名をキー、在庫数を値にした配列を作ります。
2. `foreach ($stocks as $name => $stock)` で商品名と在庫数を取り出します。
3. 在庫が0なら「売り切れ」と表示します。
4. それ以外なら在庫数を表示します。

## つまずきポイント

`$name => $stock` は、キーと値を両方取り出す書き方です。
連想配列ではよく使います。
