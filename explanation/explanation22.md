# 第22題 解説: 四角形の面積

## 考え方

四角形の面積は、幅と高さをかけて求めます。
計算処理を関数にすると、幅と高さを変えて何度でも使えます。

## 手順

1. `getArea` 関数を作ります。
2. 引数で幅と高さを受け取ります。
3. 幅と高さをかけた結果を `return` します。
4. 関数を呼び出して表示します。

## つまずきポイント

関数の中で計算した結果を外で使いたい場合は `return` します。
`echo` してしまうと、表示はできますが計算結果を別の処理で使いにくくなります。
