# 第30題 解説: Todoクラス

## 考え方

Todoには「タイトル」と「完了したかどうか」という状態があります。
この2つを `Todo` クラスにまとめると、1つのTodoを1つのオブジェクトとして扱えます。

## 手順

1. `Todo` クラスを作ります。
2. `title` と `isDone` をプロパティとして持ちます。
3. `complete` メソッドで完了状態にします。
4. `describe` メソッドで表示用の文字列を返します。
5. Todoを複数作り、完了前と完了後を表示します。

## つまずきポイント

プロパティはクラスの中にある変数です。
メソッドはクラスの中にある関数です。
Todoの状態を変える処理は、Todo自身のメソッドとして書くと整理しやすくなります。
