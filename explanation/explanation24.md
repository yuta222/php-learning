# 第24題 解説: ユーザー一覧

## 考え方

1人のユーザーには名前と年齢があります。
このような複数の項目を持つデータは、連想配列で表すと読みやすくなります。

## 手順

1. ユーザーを `name` と `age` を持つ連想配列にします。
2. 3人分を配列にまとめます。
3. `foreach` で1人ずつ表示します。
4. 年齢が20以上なら人数を1増やします。

## つまずきポイント

`$user["name"]` と `$user["age"]` のように、キー名を使って値を取り出します。
キー名は文字列なので引用符で囲みます。

## もう一度練習

ユーザーを1人追加して、20歳以上の人数が正しく変わるか確認しましょう。
