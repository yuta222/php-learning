# 第14題 解説: 割り勘計算

## 考え方

割り勘は、合計金額を人数で割る計算です。
PHPでは割り算に `/` を使います。

## 手順

1. 合計金額を `$totalPrice` に入れます。
2. 人数を `$people` に入れます。
3. `$totalPrice / $people` で1人あたりの金額を出します。
4. 結果を表示します。

## つまずきポイント

人数が0だと割り算できません。
今回は固定値なので問題ありませんが、入力値を使う場合は0でないか確認が必要です。

## もう一度練習

合計金額を5000円、人数を3人に変えて、結果がどうなるか確認してください。
