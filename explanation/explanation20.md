# 第20題 解説: 買い物リスト

## 考え方

買い物リストのように、同じ種類のデータを並べるときは配列を使います。
`foreach` を使うと、配列の中身を1つずつ表示できます。

## 手順

1. 買うものを配列に入れます。
2. `foreach` で商品名を取り出します。
3. インデックスに1を足して番号を作ります。
4. `count` で品数を表示します。

## つまずきポイント

インデックスは0から始まります。
人に見せる番号は1から始めたいので、`$index + 1` にします。

## もう一度練習

買い物リストに「チーズ」を追加して、品数が増えることを確認しましょう。
