# 第6題 解説: 配列

## 考え方

配列は、複数の値をまとめて扱うための箱です。
点数のように同じ種類の値がいくつもある場合に便利です。

## 手順

1. `$scores = [72, 88, 91, 65, 79];` のように配列を作ります。
2. `foreach` で1つずつ点数を取り出します。
3. 取り出した点数を合計に足します。
4. `count` で人数を数え、平均を出します。
5. `max` と `min` で最高点と最低点を出します。

## つまずきポイント

配列の番号は0から始まります。
表示で「1人目」としたい場合は、インデックスに1を足します。

## もう一度練習

点数を1つ増やして、合計と平均が変わることを確認しましょう。
