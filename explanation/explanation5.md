# 第5題 解説: 繰り返し

## 考え方

同じような処理を何度も行うときは、繰り返しを使います。
回数が決まっている処理は `for`、条件が続く間だけ動かす処理は `while` が使いやすいです。

## 手順

1. `for` で1から30まで数を増やします。
2. 15の倍数は3と5の両方の倍数なので、最初に判定します。
3. 次に3の倍数、5の倍数、その他を分けます。
4. `while` で1から10まで足し算します。

## つまずきポイント

FizzBuzzでは、3の倍数より先に15の倍数を判定します。
そうしないと15が `Fizz` になってしまいます。

1から50までに変えて実行してみましょう。
