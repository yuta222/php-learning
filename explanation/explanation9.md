# 第9題 解説: CLI入力と検証

## 考え方

CLIはターミナルで動かすプログラムのことです。
PHPでは `$argv` にコマンドライン引数が入ります。

## 手順

1. `$argv[1]` に点数が入っているか確認します。
2. `is_numeric` で数値かどうか確認します。
3. 0から100の範囲か確認します。
4. 正しい点数だけ評価判定へ進めます。

## つまずきポイント

入力値は最初から正しいとは限りません。
計算や判定の前に、未入力、数値以外、範囲外を確認しましょう。
