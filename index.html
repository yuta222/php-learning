使い方

問題を書き、解説を読み、解答例と比べます。

1

PHPの確認

php -v を実行します。

2

問題を解く

TODOを読んでコードを書きます。

3

解説を読む

考え方と手順を確認します。

4

解答と比べる

解答例と自分の書き方を比べます。

5

もう一度書く

同じ問題をもう一度書き直します。

実行コマンド
git clone https://github.com/yuta222/php-learning.git
cd php-learning

php -v
php question/question1.php
php answer/answer1.php

# 第10題・第28題 Webフォーム
php -S 127.0.0.1:8000 -t question

6週間の進め方

1日30分から60分を目安に、難易度を上げずに反復しながら進めます。

1週目

第1題から第5題。PHPの実行、変数、文字列、条件分岐に慣れます。

2週目

第6題から第10題。配列、関数、連想配列、CLI、Webフォームを試します。

3週目

第11題から第15題。保存、クラス、変数、計算、文字列を復習します。

4週目

第16題から第20題。条件分岐、繰り返し、配列を反復練習します。

5週目

第21題から第25題。配列、関数、連想配列の組み合わせを練習します。

6週目

第26題から第30題。CLI、フォーム、JSON、クラスをもう一度練習します。
